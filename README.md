# UET Peshawar Login Page 🎓
 
A modern and responsive login page for University of Engineering & Technology (UET) Peshawar.
 
---

## Features ✨
 
- 📱 **Fully Responsive** - Works on desktop, tablet, and mobile
- 🎨 **Modern Design** - Clean UI with UET brand colors
- ✨ **Smooth Animations** - Fade-in and slide animations
- 🎯 **User-Friendly** - Easy to use form with validation
---
 
## Colors Used 🎨
 
- **Primary**: Dark Blue `#1e3a5f`
- **Secondary**: Gold `#DAA520`
- **Background**: Blue Gradient
